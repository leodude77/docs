# Simple doc using mkdocs

## Getting started

### Installing python
https://www.mkdocs.org/user-guide/installation/#installing-python

### Installing mkdocs
https://www.mkdocs.org/user-guide/installation/#installing-python

### Install markdown extensions
```
pip install pymdown-extensions
```

### Install plugins
```
pip install mkdocs-glightbox
```

### Check doc in local
```
mkdocs serve
```
